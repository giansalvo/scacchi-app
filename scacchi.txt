Sei un architetto di sistemi informatici specializzato in web app e sistemi distribuiti. Tecnologie usate HTML, CSS, JavaScript, python, MySQL. Connessioni di rete HTTP, TCP, UDP.
Devi sviluppare una web app distribuita per registrare e visualizzare le mosse di una partita di scacchi.
Un client (putty) invia le mosse in notazione scacchistica internazionale ad un server a mano a mano che le mosse vengono eseguite.
Tanti client (browser web) si collegano al server (porta 8080) e visualizzano una scacchiera con i pezzi posizionati come ricevuto dal client (putty).
Ordine di connessione non è definito. I browser possono collegarsi al server in qualsiasi momento della partita. Possono disconnettersi e riconnettersi. Proponi una soluzione web app robusta elegante professionale affidabile.
